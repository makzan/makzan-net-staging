The process of building native app vs web app

This is the graph that I use when comparing building app with web technology (HTML5) or native IDE (Xcode + Swift or Android Studio + Java).

Using the pure web technology to build app is fast. But it could take huge amount of time to make it polish and fix every little detail to make to act as app behavior (Because it’s designed for web pages). On the other hands, using native technology to build app seemed to be slow, but in the long run, it has more stable curve to reach the polish state.
